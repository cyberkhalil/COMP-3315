📡 Ch 1 Computer Networks and the Internet Lecture 1
§1.1‑1.7 | edge, core, delay, layers, security, history

Learning Outcomes

  • Describe Internet: nuts‑and‑bolts vs service view
  • Differentiate packet switching vs circuit switching
  • Compute delay, loss, throughput in networks
  • Explain protocol layers (TCP/IP, OSI) & encapsulation
  • Identify basic network attacks (DoS, sniffing)

Book Sections

  • 1.1 What is the Internet? • 1.2 Network edge
  • 1.3 Network core • 1.4 Delay, loss, throughput
  • 1.5 Protocol layers, service models
  • 1.6 Networks under attack • 1.7 History
Wireshark Lab: “Getting Started” – capture HTTP traffic, explore headers, packet details.
🌐 Ch 2 Application Layer Lecture 2
§2.1‑2.7 | Web, email, DNS, P2P, socket API

Learning Outcomes

  • Architect client‑server vs P2P applications
  • Analyze HTTP (persistent/non‑persistent), cookies, caching
  • Understand SMTP, POP3, IMAP for email
  • Implement a basic UDP/TCP socket program
  • Explain DNS hierarchy & resource records

Book Sections

  • 2.1 Principles of app layer protocols
  • 2.2 Web and HTTP (HTTP/1.1, HTTP/2, HTTP/3 intro)
  • 2.3 FTP • 2.4 Email (SMTP, POP3)
  • 2.5 DNS • 2.6 P2P (BitTorrent)
  • 2.7 Socket programming with TCP/UDP
Socket Programming: Build a UDP ping client and a multi‑threaded HTTP server (Python). Analyze HTTP requests with Wireshark.
🚚 Ch 3 Transport Layer Lecture 3
§3.1‑3.6 | UDP, TCP, flow & congestion control

Learning Outcomes

  • Compare reliable vs unreliable transport
  • Understand multiplexing/demultiplexing
  • Explain principles of reliable data transfer (rdt)
  • Analyze TCP segment structure, flow control, and AIMD
  • Differentiate between Tahoe, Reno, CUBIC

Book Sections

  • 3.1 Transport‑layer services
  • 3.2 Multiplexing/demultiplexing
  • 3.3 Connectionless UDP
  • 3.4 Principles of reliable data transfer
  • 3.5 TCP (segment, RTT, flow control)
  • 3.6 Congestion control (AIMD, fairness)
Wireshark TCP Lab: Trace three‑way handshake, sequence/ACK numbers, window scaling, and observe congestion window evolution.
📦 Ch 4 The Network Layer: Data Plane Lecture 4
§4.1‑4.4 | IP, addressing, forwarding, SDN data plane

Learning Outcomes

  • Differentiate forwarding vs routing
  • Understand IPv4 datagram format, fragmentation
  • Perform subnet calculations (CIDR, subnet masks)
  • Explain NAT and IPv6 transition methods
  • Describe router components and generalized forwarding (SDN)

Book Sections

  • 4.1 Overview of network layer
  • 4.2 Router components & switching fabrics
  • 4.3 IP: IPv4, DHCP, NAT, IPv6
  • 4.4 Generalized Forwarding (SDN match+action)
IP & Routing Lab: Configure static routes with Cisco Packet Tracer, analyze IP fragmentation, and capture ICMP packets with Wireshark.
🕹️ Ch 5 The Network Layer: Control Plane Lecture 5
§5.1‑5.6 | routing algorithms, OSPF, BGP, SDN, ICMP

Learning Outcomes

  • Compare Link‑State (Dijkstra) vs Distance‑Vector (Bellman‑Ford)
  • Configure OSPF intra‑domain routing
  • Explain BGP inter‑domain routing (eBGP/iBGP)
  • Describe SDN control plane (OpenFlow, controllers)
  • Use ICMP, traceroute, and SNMP

Book Sections

  • 5.1 Introduction • 5.2 Routing algorithms (LS, DV)
  • 5.3 Intra‑AS routing (OSPF)
  • 5.4 Routing among ISPs (BGP)
  • 5.5 SDN control plane
  • 5.6 ICMP, SNMP, NETCONF/YANG
Routing Simulation: Build OSPF network in FRRouting or Packet Tracer; use traceroute to observe path changes; analyze BGP advertisements.
🔗 Ch 6 The Link Layer and LANs Lecture 6
§6.1‑6.6 | error detection, multiple access, Ethernet, switches, VLANs

Learning Outcomes

  • Understand error detection (CRC, checksum)
  • Compare multiple access protocols (CSMA/CD, slotted Aloha)
  • Analyze Ethernet frame & switch self‑learning
  • Configure VLANs and explain ARP protocol
  • Describe link virtualization (MPLS) and data center networks

Book Sections

  • 6.1 Link‑layer services
  • 6.2 Error detection/correction
  • 6.3 Multiple access protocols
  • 6.4 Switched LANs (ARP, Ethernet, switches, VLANs)
  • 6.5 Link virtualization (MPLS)
  • 6.6 Data center networking
Ethernet & VLAN Lab: Build switched LAN with VLAN segmentation; capture ARP broadcasts and observe spanning‑tree (STP) with Wireshark.
📡 Ch 7 Wireless and Mobile Networks Lecture 7
§7.1‑7.5 | WiFi, cellular, mobility principles

Learning Outcomes

  • Characterize wireless links (SNR, hidden terminal)
  • Explain 802.11 MAC (CSMA/CA) and frame structure
  • Describe cellular generations (4G LTE, 5G NR)
  • Understand mobility: indirect/direct routing, Mobile IP

Book Sections

  • 7.1 Introduction • 7.2 Wireless links characteristics
  • 7.3 WiFi: 802.11 (association, CSMA/CA)
  • 7.4 Cellular networks (4G/5G, core network)
  • 7.5 Mobility management (home network, care‑of‑address)
WiFi Analysis: Capture 802.11 beacon frames and probe requests using Wireshark in monitor mode; examine authentication/association.
🔐 Ch 8 Security in Computer Networks Lecture 8
§8.1‑8.9 | crypto, SSL/TLS, IPsec, wireless security, firewalls

Learning Outcomes

  • Understand symmetric/asymmetric encryption & message integrity
  • Explain SSL/TLS handshake and record protocol
  • Configure IPsec VPN (ESP/AH) basics
  • Differentiate firewalls (stateless/stateful) and IDS/IPS
  • Describe 802.11i (WPA2) and operational security

Book Sections

  • 8.1 What is network security? • 8.2 Principles of cryptography
  • 8.3 Message integrity (MAC, digital signatures)
  • 8.4 End‑point authentication
  • 8.5 Securing email (PGP) • 8.6 Securing TCP (SSL/TLS)
  • 8.7 Network‑layer security (IPsec, VPN)
  • 8.8 Securing wireless LANs (WPA2, 802.11i)
  • 8.9 Operational security (firewalls, IDS/IPS)
Security Lab: Inspect TLS handshake with Wireshark, decode X.509 certificates; configure a simple IPsec tunnel between two hosts.
